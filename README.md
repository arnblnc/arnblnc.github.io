A personal portfolio website showcasing my experience and projects as a Robotics & Mechanical Engineer. Designed to highlight cross-disciplinary expertise bridging the gap between hardware and software.

## 🛠️ Tech Stack
This is a lightweight, static, single-page application built for maximum performance and accessibility without heavy framework overhead.
* **Structure:** HTML5
* **Styling:** Tailwind CSS (via CDN)
* **Interactivity:** Vanilla JavaScript
* **Hosting:** GitHub Pages
* **Forms:** Formspree API

## 📂 Repository Structure
* `index.html`: The core website source code and UI logic.
* `AaronBlanco-CV.pdf`: The downloadable professional CV linked directly to the site's hero and experience sections.
* `pfp-ab.jpeg`: Compressed profile asset.

## 🚀 Local Development
Because this site operates without a backend or build step, testing it locally is extremely straightforward:
1. Clone the repository: `git clone https://github.com/arnblnc/arnblnc.github.io.git`
2. Open the directory and double-click `index.html` to view it in any modern browser.
